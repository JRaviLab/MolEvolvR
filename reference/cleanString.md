# Clean String

Keep only alphanumerics, "*", "+", and "." in strings and substitute
spaces with "*". Used in MolEvolvR codebase to cleanup domain
architecture values

## Usage

``` r
cleanString(string)
```

## Arguments

- string:

## Value

[rlang::string](https://rlang.r-lib.org/reference/string.html) string
with only alphanumerics, "\_", "+", and "."

## Examples

``` r
if (FALSE) { # \dontrun{
cleanString()
} # }
```
