# Cleanup Lineage

Cleanup Lineage

## Usage

``` r
cleanLineage(prot, lins_rename)
```

## Arguments

- prot:

  A data frame containing a 'Lineage' column that needs to be cleaned
  up.

- lins_rename:

  A data frame with two columns: 'old' containing terms to be replaced
  and 'new' containing the corresponding replacement terms.

## Value

The original data frame with the 'Lineage' column updated based on the
provided replacements.

## Examples

``` r
if (FALSE) { # \dontrun{
cleanLineage()
} # }
```
