# Cleanup GeneDesc

Cleanup GeneDesc

## Usage

``` r
cleanGeneDescription(prot, column)
```

## Arguments

- prot:

  A data frame containing the gene descriptions.

- column:

  The name of the column from which gene descriptions are pulled for
  cleanup.

## Value

Return trailing period that occurs in GeneDesc column

## Examples

``` r
if (FALSE) { # \dontrun{
cleanGeneDescription()
} # }
```
