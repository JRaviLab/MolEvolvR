# Format job arguments into html-formatted key/value pairs, for including in an email

Format job arguments into html-formatted key/value pairs, for including
in an email

## Usage

``` r
formatJobArgumentsHTML(job_args)
```

## Arguments

- job_args:

  a list of job arguments, e.g. as read from the job_args.yml file

## Value

a list of HTML-formatted key/value pairs

## Examples

``` r
if (FALSE) { # \dontrun{
formatJobArgumentsHTML("/data/scratch/janani/molevolvr_out/Ba5sV1_full")
} # }
```
