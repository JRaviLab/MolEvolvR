``` r
readIPRScanTSV(filepath)
```

- filepath:

  [rlang::chr](https://rlang.r-lib.org/reference/vector-construction.html)
  path to interproscan output TSV

## Value

[tbl_df](https://dplyr.tidyverse.org/reference/tbl_df.html) interproscan
output table
