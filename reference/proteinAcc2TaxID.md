# proteinAcc2TaxID

proteinAcc2TaxID

## Usage

``` r
proteinAcc2TaxID(accnums, suffix, out_path, return_dt = FALSE)
```

## Arguments

- accnums:

  A character vector of protein accession numbers to be mapped to
  TaxIDs.

- suffix:

  A string suffix used to name the output file generated by the script.

- out_path:

  A string specifying the directory where the output file will be saved.

- return_dt:

  A logical indicating whether to return the result as a data table.
  Defaults to FALSE. If TRUE, the output file is read into a data table
  and returned.

## Value

If `return_dt` is TRUE, a data table containing the mapping of protein
accession numbers to TaxIDs. If FALSE, the function returns NULL.

## Examples

``` r
if (FALSE) { # \dontrun{
# Example accession numbers
accessions <- c("ABC123", "XYZ456", "LMN789")
tax_data <- proteinAcc2TaxID(accessions, suffix = "example",
out_path = "/path/to/output", return_dt = TRUE)
tax_data
} # }
```
